## 说明

本脚本提供了bitahub的显卡仪表盘，方便大家训练时查看可用显卡，以更充分地利用算力资源。

- 仪表盘的入口在**页面右下角**，将原来的咨询按钮换为了仪表盘按钮；
- 每次**鼠标移到图标上**时，将会查询一次可用显卡信息，并显示在左侧；
- 若要刷新信息，可以将鼠标**移出再移入**，或者点击表格左上角的**Refresh**按钮；
